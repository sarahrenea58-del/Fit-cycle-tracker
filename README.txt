GitHub Pages upload contents

1. Create a new public repository in GitHub, for example: fit-cycle-tracker
2. Upload all files from this folder to the root of that repository
3. In GitHub, open Settings > Pages
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save
6. Wait about 1-3 minutes
7. Your site will appear at:
   https://YOUR-USERNAME.github.io/REPO-NAME/

Files included:
- index.html
- .nojekyll

On iPhone:
- Open the GitHub Pages URL in Safari
- Tap Share > Add to Home Screen
